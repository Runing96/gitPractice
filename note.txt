1.创建本地仓库
创建文件夹，并初始化：git init(会生成一个.git文件夹)

2.远程仓库的创建
1)创建项目

3.本地环境与SSH配置
1)ssh-keygen命令查看公钥
2)点击github个人头像->settings->SSH and GPG keys->new SSH key->复制公钥
3)本地环境配置
   git config Cglobal user.name ‘xxx’
   git config Cglobal user.email   example@xx.com
4)本地分支与远程分支建立关联
git remote add origin yougithubAdress(your profile->具体的仓库)

4.从远程仓库拉取分支：
git pull origin master
